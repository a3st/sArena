# Enhanced Arena Frames (Cata Classic)

## Features
- Trinket & Racial Tracker
- DR Tracker
- See important buffs & debuffs
- See interrupt lockouts
- Multiple layouts
- Enemy spec detections

Ctrl+Shift+Click to drag stuff
